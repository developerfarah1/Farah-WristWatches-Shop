# Online Shopping Cart (Java MVC)

## Project Description
This is a web-based online shopping cart application developed using Java MVC architecture.
It allows users to register, log in, browse products, add items to a cart, and see the total price.

## Features
- User Registration & Login
- Product Catalog with Name, Price, and Description
- Add / Remove items from Cart
- Session-based Cart Management
- Dynamic Total Price Calculation

## Technologies Used
- Java
- JSP & Servlets
- MVC Architecture
- MySQL Database
- Apache Tomcat

## Purpose
Educational project to demonstrate Java Web MVC concepts and basic e-commerce functionality.
